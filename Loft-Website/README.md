# Loft Mobile Golf Studio - Landing Page

A modern, minimalist landing page for Loft Mobile Golf Studio built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, add your Loft logo file to `public/assets/loft-logo.png` (or update the path in `app/page.tsx`).

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/page.tsx` - Main landing page component
- `app/layout.tsx` - Root layout with metadata and fonts
- `app/globals.css` - Global styles and CSS variables
- `public/assets/` - Static assets (logo, images)

## Brand Colors

- **Loft Green**: `#1B4D3E` (dark forest green)
- **Loft Off-White**: `#F5F5F0` (off-white)

## Typography

- **Headings**: Playfair Display (serif)
- **Body**: Inter (sans-serif)

## Build

```bash
npm run build
npm start
```
